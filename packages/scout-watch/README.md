# @4meta5/scout-watch

Experimental watch/review features for Scout CLI.

**Warning: Not part of core guarantees. May change without notice.**

## Installation

```bash
npm install @4meta5/scout-watch
```

This package provides watch and review capabilities for Scout. It requires `@4meta5/scout` as a peer dependency.

## Features

- Track validated repos over time
- Generate differential review sessions
- Detect drift in tracked paths
- Chunk large diffs for manageable review

## Usage

See the [Scout documentation](https://github.com/4meta5/scout#experimental-features) for usage instructions.
